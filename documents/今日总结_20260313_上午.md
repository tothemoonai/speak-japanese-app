## 🔧 技术要点

### Kotlin data class 与 Java 互操作

**问题**：Kotlin 定义的 data class 在 Java 中需要特定的构造器

**解决方案**：
```kotlin
// Kotlin 定义
data class OfflineSenseVoiceModelConfig(
    var model: String = "",
    var language: String = "",
    var useInverseTextNormalization: Boolean = true,
    var qnnConfig: QnnConfig = QnnConfig(),
)

// Java 中调用（必须提供所有参数）
OfflineSenseVoiceModelConfig config = new OfflineSenseVoiceModelConfig(
    modelPath,
    language,
    useItn,
    new QnnConfig()  // 即使有默认值也必须提供
);
```

### Capacitor 本地插件注册

在 Capacitor 6.x 中，本地插件注册需要：

1. **注解**：`@CapacitorPlugin(name = "LocalASR")`
2. **JSON 配置**：`capacitor.plugins.json`
3. **显式注册**：`MainActivity.registerPlugin(PluginClass.class)`
