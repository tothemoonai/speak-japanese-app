## 为什么选择智谱GLM

### 优势

1. **国内访问速度快**
   - 服务器部署在国内
   - 低延迟，高稳定性
   - 无需翻墙

2. **价格实惠**
   - GLM-3-Turbo：输入 ¥0.005/千tokens，输出 ¥0.005/千tokens
   - GLM-4：输入 ¥0.50/千tokens，输出 ¥2.00/千tokens
   - 新用户有免费试用额度

3. **中文理解能力强**
   - 对中文语境理解深入
   - 同时支持日语评估

4. **API兼容性好**
   - 兼容OpenAI API格式
   - 易于集成

### 成本对比

假设每次练习评估10句话，约消耗1000 tokens：

| 模型 | 每次成本 | 100次成本 | 1000次成本 |
|------|---------|----------|-----------|
| GLM-3-Turbo | ¥0.005-¥0.01 | ¥0.50-¥1 | ¥5-¥10 |
| GLM-4 | ¥0.25-¥0.50 | ¥25-¥50 | ¥250-¥500 |
| GPT-4 | ~$0.03 | ~$3 | ~$30 |

## 注册与认证

### 步骤1: 访问智谱AI开放平台

打开浏览器，访问：https://open.bigmodel.cn/

### 步骤2: 注册账号

1. 点击右上角 "注册" 按钮
2. 使用手机号或邮箱注册
   - 手机号注册：需要接收验证码
   - 邮箱注册：需要验证邮箱
3. 设置密码并完成注册

### 步骤3: 实名认证

⚠️ **重要**：使用API功能需要完成实名认证

1. 登录后进入控制台
2. 点击 "实名认证"
3. 准备材料：
   - 本人身份证
   - 手机号（用于接收验证码）
4. 填写信息：
   - 真实姓名
   - 身份证号
   - 上传身份证正反面照片
5. 人脸识别（可能需要）
6. 等待审核（通常几分钟到1小时）

## 获取API密钥

### 步骤1: 进入API密钥管理

1. 登录后进入控制台
2. 左侧菜单找到 "API密钥"
3. 点击进入密钥管理页面

### 步骤2: 创建新的API密钥

1. 点击右上角 "创建新的API密钥" 按钮
2. 弹窗中可以：
   - 输入密钥名称（可选，如 "日语口语App"）
   - 选择过期时间（建议选择 "永不过期"）
3. 点击 "确认创建"

### 步骤3: 保存密钥

⚠️ **重要**：密钥只显示一次，请务必保存！

密钥格式类似：
```
xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx.xx.xxxxxxxx
```

包含两部分，用 `.` 分隔：
- 第一部分：API ID
- 第二部分：API Secret

## 配置项目

### 步骤1: 打开环境变量文件

在项目根目录下，找到 `.env.local` 文件。如果没有，从 `.env.example` 复制：

```bash
cp .env.example .env.local
```

### 步骤2: 添加智谱API密钥

在 `.env.local` 文件中添加：

```env
# 智谱GLM配置
ZHIPU_API_KEY=你的API密钥
```

示例：
```env
ZHIPU_API_KEY=1234567890abcdef1234567890abcdef.abcdef1234567890
```

### 步骤3: 选择模型

智谱提供多个模型版本，推荐配置：

```env
# 可选：指定要使用的GLM模型（默认使用glm-4）
ZHIPU_MODEL=glm-4-flash
# 或使用更便宜的GLM-3-Turbo
# ZHIPU_MODEL=glm-3-turbo
```

**模型说明**：

| 模型 | 特点 | 价格 | 推荐场景 |
|------|------|------|---------|
| glm-4-flash | 最新版本，速度快 | ¥0.50/千tokens | 生产环境（推荐） |
| glm-4 | 标准版本 | ¥0.50/千tokens | 高质量评估 |
| glm-3-turbo | 速度快，价格低 | ¥0.005/千tokens | 测试开发 |

## 常见问题

### Q1: API密钥无效

**错误信息**：`Zhipu API key is not configured`

**解决方案**：
1. 检查 `.env.local` 文件是否存在
2. 确认 `ZHIPU_API_KEY` 已正确配置
3. 检查密钥格式是否正确
4. 重启开发服务器

### Q2: 认证失败

**错误信息**：`Unauthorized (401)`

**可能原因**：
1. API密钥不正确
2. API密钥已过期
3. 账户余额不足

**解决方案**：
1. 重新复制API密钥
2. 登录智谱控制台检查账户状态
3. 充值或检查免费额度

### Q3: 实名认证未通过

**可能原因**：
1. 身份证信息填写错误
2. 照片不清晰
3. 系统审核中

**解决方案**：
1. 仔细核对身份证信息
2. 确保照片清晰、完整
3. 耐心等待审核（通常1小时内）

### Q4: 额度不足

**错误信息**：`Insufficient balance`

**解决方案**：
1. 登录智谱控制台查看余额
2. 充值账户
3. 或使用更便宜的GLM-3-Turbo模型

### Q5: 网络超时

**错误信息**：`Network timeout` 或 `ETIMEDOUT`

**解决方案**：
1. 检查网络连接
2. 确认防火墙设置
3. 尝试使用VPN（如果网络受限）
4. 检查代理设置

### Q6: 日语评估不准确

**原因**：虽然GLM-4支持日语，但主要优化在中文

**解决方案**：
1. 在提示词中明确要求日语评估
2. 可以考虑使用日本模型（如OpenAI GPT-4）
3. 调整评估提示词模板

4. 实现请求缓存

### 实现重试机制

```typescript
async function evaluateWithZhipuRetry(input: EvaluationInput, maxRetries = 3) {
  for (let i = 0; i < maxRetries; i++) {
    try {
      return await evaluateWithZhipu(input);
    } catch (error) {
      if (i === maxRetries - 1) throw error;
      // 指数退避
      await new Promise(resolve => setTimeout(resolve, Math.pow(2, i) * 1000));
    }
  }
}
```
