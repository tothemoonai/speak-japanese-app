# Android ASR 测试清单

## 📋 **测试前准备**

### 环境检查

- [ ] Android SDK 已安装
- [ ] Android 设备或模拟器已连接
- [ ] USB 调试已启用
- [ ] ADB 可以正常连接设备

### 模型文件检查

- [ ] `model.int8.onnx` (227 MB) 存在于 `assets/models/`
- [ ] `tokens.txt` (309 KB) 存在于 `assets/models/`
- [ ] JNI 库存在于 `jniLibs/` 对应架构目录

---

## 🏗️ **构建测试**

### 步骤 1: 构建 APK

```bash
# Windows
cd android
gradlew.bat assembleDebug

# Linux/Mac
cd android
./gradlew assembleDebug
```

- [ ] 构建成功，无错误
- [ ] APK 文件生成: `app/build/outputs/apk/debug/app-debug.apk`
- [ ] APK 大小约为 250-300 MB（包含模型）

### 步骤 2: 安装到设备

```bash
# 使用 ADB 安装
adb install android/app/build/outputs/apk/debug/app-debug.apk

# 或使用脚本
scripts/build-android.bat
```

- [ ] 安装成功
- [ ] 应用图标显示在主屏幕
- [ ] 应用可以正常启动
