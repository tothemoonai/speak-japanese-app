## 🐛 发现并修复的Bug汇总

### 1. Report Service 关键Bug

**文件**: `src/services/supabase/report.service.ts`

#### Bug 1: bestScore 变量未定义
- **位置**: Line 254
- **问题**: 当practices为空数组时，Math.max返回-Infinity，导致bestScore为undefined
- **修复**: 添加了空数组检查
```typescript
// 修复前
const bestScore = Math.max(...practices.map((p) => p.total_score || 0));

// 修复后
const bestScore = practices.length > 0
  ? Math.max(...practices.map((p) => p.total_score || 0))
  : 0;
```

#### Bug 2: 返回对象变量引用错误
- **位置**: Line 269
- **问题**: 使用了简写属性`best_score,`但变量名为`bestScore`
- **修复**: 改为显式属性赋值
```typescript
// 修复前
best_score,

// 修复后
best_score: bestScore,
```

#### Bug 3: 空周报返回空进度图
- **位置**: Lines 106-121
- **问题**: 当没有练习记录时，返回空数组而不是7天进度图
- **修复**: 调用createWeeklyProgressChart生成完整的7天图
```typescript
// 修复后
if (!practices || practices.length === 0) {
```

### 4. useCourse Hooks 性能问题

**文件**: `src/hooks/useCourse.ts`

#### Bug: refetch函数引用不稳定
- **影响**: 所有useCourse相关hooks
- **问题**: fetch函数定义在useEffect内部，导致返回的refetch函数引用不稳定
- **修复**: 重构为useCallback模式
```typescript
// 修复前
export function useCourses(userId?: string) {
  const [data, setData] = useState<CourseWithProgress[] | null>(null);
  const [error, setError] = useState<PostgrestError | null>(null);
  const [isLoading, setIsLoading] = useState(true);

  useEffect(() => {
    async function fetchCourses() {
      setIsLoading(true);
      const { data: courses, error: err } = await courseService.getAllCourses(userId);
      setData(courses);
      setError(err);
      setIsLoading(false);
    }

    fetchCourses();
  }, [userId]);

  return { data, error, isLoading, refetch: () => fetchCourses() }; // BUG: fetchCourses不在作用域
}

// 修复后
export function useCourses(userId?: string) {
  const [data, setData] = useState<CourseWithProgress[] | null>(null);
  const [error, setError] = useState<PostgrestError | null>(null);
  const [isLoading, setIsLoading] = useState(true);

  const fetchCourses = useCallback(async () => {
    setIsLoading(true);
    const { data: courses, error: err } = await courseService.getAllCourses(userId);
    setData(courses);
    setError(err);
    setIsLoading(false);
  }, [userId]);

  useEffect(() => {
    fetchCourses();
  }, [fetchCourses]);

  return { data, error, isLoading, refetch: fetchCourses }; // 稳定的函数引用
}
```
