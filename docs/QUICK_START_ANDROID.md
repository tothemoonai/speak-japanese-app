# Android ASR 快速测试指南

## 📱 安装和快速测试

### 1. 安装更新版APK

**APK文件**：`android/app/build/outputs/apk/debug/app-debug.apk`
**编译时间**：2026-03-12 17:56
**文件大小**：265 MB

**安装方法**：

```bash
# 方法A：使用ADB（推荐）
adb uninstall com.speakjapanese.app
adb install android/app/build/outputs/apk/debug/app-debug.apk

# 方法B：直接传输
# 将APK文件复制到设备，直接点击安装
```

### 2. 访问测试页面

1. **打开APP**
2. **登录账户**
3. **在Dashboard底部**点击"📱 Android ASR 测试"链接
4. **或直接输入URL**：`https://speak-japanese-app.vercel.app/test-android-asr`

### 3. 测试ASR功能

**步骤**：
1. 点击"初始化 ASR"按钮
2. 等待初始化完成（首次10-30秒）
3. 看到"就绪"状态
4. 点击"开始录音"
5. 授予麦克风权限
6. 说日语句子（如"こんにちは"）
7. 点击"停止录音"
8. 查看识别结果

## 🔍 如果初始化失败

### 快速诊断（使用ADB查看日志）

```bash
# 1. 查看实时日志
adb logcat | grep -E "LocalASR|SherpaASR"

# 2. 清除旧日志并开始新会话
adb logcat -c
adb logcat | grep -E "LocalASR|SherpaASR"

# 3. 保存完整日志
adb logcat -v time > asr_debug.log
```

### 常见问题及快速解决

#### 问题1：提示"初始化失败"

**可能原因**：
- 模型文件复制失败
- JNI库加载失败
- 存储空间不足

**快速解决**：
1. 设置 → 应用 → 日语口语练习 → 清除数据
2. 重新打开APP并尝试
3. 确保有至少500MB可用存储空间

#### 问题2：没有麦克风权限

**解决**：
设置 → 应用 → 日语口语练习 → 权限 → 麦克风（允许）

#### 问题3：识别结果不准确

**建议**：
- 在安静环境测试
- 清晰地发音
- 说完整的句子（3-10秒）
- 测试常用词汇：
  - こんにちは（你好）
  - ありがとう（谢谢）
  - さようなら（再见）
